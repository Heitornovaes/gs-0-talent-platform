# gs-0-talent-platform

Plataformas que conectam talentos a projetos com propósito.

## Membros do grupo

- Gustavo Matos  
- Heitor Novaes  

## Descrição do tema

Plataforma digital que conecta profissionais a projetos alinhados a causas
sociais, ambientais ou de impacto positivo, permitindo que organizações
encontrem talentos com habilidades adequadas e interesse no propósito.

## Tecnologias

- Java 17  
- Spring Boot 3  
- Maven  
- Swagger / OpenAPI  
- Docker

```bash
./mvnw spring-boot:run
